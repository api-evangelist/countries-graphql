# Countries GraphQL API - Plans

## Overview

The Countries GraphQL API is a free, open-source public API maintained by Trevor Blades. There are no commercial plans or paid tiers — the API is available to all users at no cost.

## Free Plan

- **Cost:** Free
- **Access:** Unrestricted public access
- **Endpoint:** https://countries.trevorblades.com
- **Features:**
  - Query countries, continents, and languages
  - Filter countries by code, currency, or continent
  - Access capital cities, currencies, phone codes, emoji flags, languages, and AWS region data
  - Access administrative subdivisions (provinces/states)

## Notes

No sign-up, API key, or authentication is required to use the API. The project is MIT-licensed and hosted as a community service. Pricing information is not publicly documented because the API has no paid tier.

The API is powered by Stellate for GraphQL API management and caching.
